计算GNP与平均寿命和人口的相关性

使用corr函数计算相关性，越接近 0，表示越没有相关性。越接近 1，则正相关性越强，越接近 -1，则负相关性越强：GNP与人口的相关性比平均寿命相关性强。

在多变量因素影响时比较有用，例如成绩，房价受多方面因素影响。
